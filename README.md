# Titanic_Survival_Analysis_EDA_Features_Engineering_Pandas_Colab

A comprehensive data science project utilizing the Pandas library to clean, transform, and analyze the Titanic dataset. This project focuses on identifying key factors—such as socio-economic status, age, and gender—that influenced passenger survival rates.

Check Assignment file for details

1. Handle missing data
2. Create new features
3. Analyze patterns
4. Prepare data for machine learning
